利用ngrok搭建外网可以访问的本地服务器

微信公众号开发的时候，由于在自己电脑上开发，而微信接入需要一个外网的域名地址，用ngrok搭建一个反向代理就非常方便了。

使用方法：

注册 -> 开通一个隧道 -> 利用隧道的id -> 启动‘Sunny-Ngrok启动工具.bat’ -> 输入隧道id -> ok;

前提要打开本地服务器；

免费的隧道有时候慢得很…
